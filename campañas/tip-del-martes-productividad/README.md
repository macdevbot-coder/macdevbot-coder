# Campaña "Tip del Martes" – Prompt estructurado para ahorrar 2 horas de correos

![Panel de productividad neon](./assets/dashboard-productividad.jpg)

## Resumen del artículo
- **Tema:** Cómo usar un prompt de IA estructurado para procesar correos y recuperar 2 horas al día.
- **Formato:** Artículo interactivo con tabs, checklist y ritual semanal.
- **Regalo:** Prompt copiable dentro de `<pre><code>` listo para usar en ChatGPT/Claude/Gemini.

Accedé al artículo completo en GitHub Pages: https://macdevbot-coder.github.io/macdevbot-coder/campañas/tip-del-martes-productividad/

## Prompt destacado
```
Actuá como mi Chief of Inbox para gestionar correos.
Contexto del día: {prioridades principales}.
Rol y voz: director/a de producto en {empresa}, tono humano, directo, positivo.
Tareas:
1. Resume cada correo en 2 líneas con intención + urgencia.
2. Propone la respuesta ideal en 120 palabras máximo, usando viñetas si hay múltiples decisiones.
3. Indica qué info falta y redacta la pregunta puntual.
Formato final:
- "Resumen": ...
- "Respuesta sugerida": ...
- "Siguiente acción": ... (elige entre Enviar / Delegar / Agendar / Archivar)
Requisitos: preserva saludos en español rioplatense, evita tecnicismos innecesarios y marca cualquier deadline con emoji ⏰.
```
